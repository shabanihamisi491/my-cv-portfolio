# my-cv-portfolio

A sleek, responsive personal cv portfolio website designed with a **Network Administrator** aesthetic. This project features a multi-page portal experience with a custom "Rainy Drop" glassmorphism theme and dynamic networking-themed backgrounds.

## 🚀 Live Demo
[https://shabanihamisi491.github.io/my-cv-portfolio/]

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure for high accessibility.
*   **CSS3:** Custom variables for theme switching, Glassmorphism UI, and Flexbox/Grid layouts.
*   **JavaScript (Vanilla):** Single-Page Application (SPA) logic for smooth page transitions and theme rotation.

## ✨ Features
*   **Single-Button Dashboard:** A centralized portal to access specific data categories.
*   **Dynamic Theming:** Switch between **Midnight**, **Light**, and a specialized **Rainy Drop** theme with high-gloss edges.
*   **Tech-Focused UI:** Custom computer networking wallpaper and system-themed iconography.
*   **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 📂 Project Structure
*   `index.html`: The core file containing the structure, styling, and navigation logic.
*   `my-photo.jpg`: Profile image used on the landing page.

## 📝 Sections Included
1.  **Personal Details:** Professional overview and role.
2.  **Educational Background:** Detailed history from Primary to University (UDSM-COICT).
3.  **Hobbies:** Highlighting interests in Dancing, Music, and Systems/Networking.
4.  **Location:** Current base in Dar es Salaam.
5.  **Contact:** Integrated Email and WhatsApp connectivity.
